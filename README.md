# homer_pico
MicroPython scripts running on HomeR's cerebellum: Raspberry Pi Pico 2

## Usage
1. Setup your Pico board with the (latest) MicroPython firmware.
2. Upload everything in the [upython_scripts](./upython_scripts/) directory to your Pico board.

> [!NOTE]
> If you are completely new, please follow the official [guide](https://projects.raspberrypi.org/en/projects/getting-started-with-the-pico/) to setup your Pico board.
